## v1.0.0 — 2025年（首发版本）

### 首次发布

**新增**：
- 4阶段工作流（SOP解析→任务拆解→手册生成→交接包输出）
- Phase/Task/Step 三层结构（P1-T2-S3唯一编号）
- 断点恢复机制（.project_state 状态文件，每 Phase 末尾保存）
- PERT 三点估时法（乐观/正常/悲观，期望值计算）
- 三不原则（不过度设计/不静默调整SOP决策/不跳过验证步骤）
- 幻觉防护：版本号/API限制强制标注 [需用户核实]
- 标准化交接包（传给 Skill 05 或用户）

**技术规格**：
- SKILL.md description：389字符（合规，≤1024）
- 触发词：中文7个，英文6个
- 工程包文件数：10个（v1.0.1后：11个）
